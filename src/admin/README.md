# Admin Panel

Эта папка предназначена для будущей административной панели.

## Планируемая структура

```
admin/
├── components/     # Переиспользуемые компоненты админки
├── pages/          # Страницы админки (Dashboard, Users, Cards, etc.)
├── hooks/          # Кастомные хуки
└── utils/          # Утилиты и хелперы
```
